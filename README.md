## Installation

### Run Locally

Clone the project

```bash
  git clone https://github.com/Chechutech/shopping_midori.git
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
 npm run dev
```

### Run Tests
```bash
 cd folder
```
```bash
 npm run test
```
